### Dependencias

Esta configuracion utiliza las siguientes paquetes

- [i3wm](https://i3wm.org/downloads/)

- [Polybar](https://github.com/polybar/polybar)

- [feh](https://feh.finalrewind.org/) (wallpapers)

- [picom](https://github.com/yshui/picom) (compositor)

- [alacritty](https://alacritty.org/) / [kitty](https://sw.kovidgoyal.net/kitty/) (terminal)

---

### Instalación

#### Clona este repositorio

```bash
git clone git@github.com:wh1te-fox/i3wm-config.git
```
#### Copiar la configuración

```bash
cd i3wm-config
cp -r i3 ~/.config/ 
```
Reinicia i3 con `Mod + Shift + R` para aplicar los cambios.

---

### Atajos principales

| Atajo | Acción |
|------|-------|
| Mod + Enter | Abrir terminal |
| Mod + D | Rofi |
| Mod + Shift + Q | Cerrar ventana |
| Mod + Shift + R | Reiniciar i3 |
| Mod + Shift + E | Salir de i3 |
